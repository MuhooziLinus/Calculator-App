# Calculator App

This is a simple command-line calculator application written in Python that performs basic arithmetic operations: addition, subtraction, multiplication, and division.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/) with division by zero protection

## Usage

1. Run the calculator:
   ```
   python calculator.py
   ```

2. Follow the prompts:
   - Enter the operation (+, -, *, /)
   - Enter the first number
   - Enter the second number
   - View the result

3. To quit, enter 'q' when prompted for operation.

## Requirements

- Python 3.x

## Example

```
Simple Calculator
Operations: + (add), - (subtract), * (multiply), / (divide)
Enter 'q' to quit
Enter operation (+, -, *, /): +
Enter first number: 5
Enter second number: 3
Result: 8.0
Enter operation (+, -, *, /): q
```
